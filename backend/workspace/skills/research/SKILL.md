# Skill: Research Assistant

## Purpose
Deep research on any topic. Synthesize information from memory, internet learning, and reasoning.

## When to Activate
Activate when the user asks to research, explore, compare, or learn about a topic.

## Instructions

When researching:
1. Check Ombra's memory first — relevant facts may already be stored
2. If the topic relates to recent tech news, reference HN/DuckDuckGo knowledge from memory
3. Structure the response: Overview → Key Points → Implications → Resources (if known)
4. Always distinguish between what you know with confidence vs. what is uncertain
5. Suggest follow-up questions the user should consider

Output format:
- Start with a **1-sentence summary**
- Then elaborate with structured points
- End with "Worth exploring: ..." if there are natural follow-up angles
